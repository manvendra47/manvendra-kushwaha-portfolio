# Manvendra Kushwaha - Personal Portfolio

A professional, responsive portfolio website showcasing my skills, experience, projects, and achievements. Built with modern web technologies that prioritize performance and clean aesthetics.

## 🚀 Features

* **Modern Design:** Clean, "Professional Polish" theme with elegant typography and whitespace.
* **Responsive Layout:** fully functional and visually appealing across desktop, tablet, and mobile devices.
* **Component-Based Architecture:** Modular React components for easy maintenance and scaling.
* **Dynamic Data:** All portfolio data (projects, skills, education) is centralized in `src/data.ts` for quick updates without digging into UI code.
* **Smooth Animations:** Integrated page transition and scroll animations using Framer Motion.

## 💻 Tech Stack

* **Frontend Framework:** React 19
* **Build Tool:** Vite
* **Styling:** Tailwind CSS (v4)
* **Animations:** Framer Motion (`motion/react`)
* **Icons:** Lucide React

## 🛠️ Running Locally

Follow these steps to run the portfolio on your local machine:

1. **Clone or Download the Repository:**
   Navigate into the project directory.

2. **Install Dependencies:**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```

.

5. **Build for Production:**
   To generate a production-ready build in the `dist` folder:
   ```bash
   npm run build
   ```

## 📝 Customization

All personal configurations, contact links, and content blocks are managed in a single file:
* **`src/data.ts`**: Update your email, social links, resume bullet points, new projects, and skills here. The UI will automatically reflect your changes.
